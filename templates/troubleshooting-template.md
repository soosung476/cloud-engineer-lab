# Troubleshooting Template

## 문제 상황
무슨 문제가 발생했는지 작성

예:
- 브라우저에서 웹 페이지 접속 불가
- SSH 접속 실패
- nginx 서비스가 실행되지 않음

---

## 증상
사용자 입장에서 보이는 현상 정리

예:
- 타임아웃 발생
- Connection refused
- Permission denied (publickey)

---

## 확인한 항목
1. 인스턴스 상태
2. Security Group 규칙
3. 서비스 상태
4. 포트 리스닝 여부
5. 로그
6. 파일 권한 / 경로
7. 사용자명 / 키 파일

---

## 원인
실제 원인이 무엇이었는지 작성

---

## 해결 방법
어떻게 해결했는지 작성

---

## 배운 점
