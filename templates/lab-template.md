# 제목

## 1. 실습 목적
이번 실습에서 무엇을 확인할 것인지 작성

e.g 본 문서는 여러 마크다운 파일의 구조적 통일을 위해 작성하고, 새 문서를 작성할 때 현 문서의 체계를 따라가도록 한다.

## 2. 실습 환경
생성한 리소스, 사용한 서비스, 주요 설정값 정리

## 3. 실습 목표
실습을 통해서 확인하고 싶은 체크포인트.

## 4. 작업 과정
순서대로 수행한 작업 작성

## 5. 확인 결과
성공 여부, 접속 결과, 로그 결과 정리

## 6. 운영 관점에서 중요한 이유
해당 실습 내용이, 클라우드 운영 관점에서 중요한 이유

## 7. 문제 발생 가능 포인트
어디서 실패할 수 있는지 작성

## 6. 트러블슈팅
실제로 발생한 문제와 해결 방법 정리

## 7. 배운 점
운영 관점에서 무엇을 배웠는지 정리
